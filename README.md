# Подписка для обхода белых списков от zieng2

Все четыре файла имеют один источник и обновляются одновременно.

## Вариации подписки

**1. vless_lite.txt** — краткая версия подписки универсального формата, содержащая только подсети, которые проверенно работают во время ограничений связи. Добавляется в большинство приложений. **РЕКОМЕНДУЕТСЯ К ИСПОЛЬЗОВАНИЮ!**

**2. vless_universal.txt** — полная версия подписки универсального формата. Содержит все доступные подсети.

**3. vless.txt** — **устаревшая** версия подписки, не добавляющаяся в v2rayN. Использовать **не** рекомендуется. В будущем поддержка будет прекращена.

**4. vless_nolite.txt** — версия подписки исключительно для тестирования. Содержит все подсети, кроме проверенно рабочих. Если есть желание помочь в разработке, можете скидывать рабочие у вас профили в Issues.

## Советы и интересная информация о подписке

**1.** Рекомендуется глобально включить опцию 'Разрешить небезопасные' в своем клиенте.

**2.** В некоторые клиенты подписка не добавится - это ожидаемо. Также, по отзывам в этой теме, в зависимости от клиента отличаются рабочие сервера. Попробуйте разные клиенты, все они активно обсуждаются в этой теме!

**3.** Источниками серверов выступают различные публичные и непубличные источники. Если у вас есть рабочие конфиги и вы хотите поделиться ими, напишите мне, пожалуйста!
Благодарность за предоставление конфигураций: ts1 (t.me/ByeWhiteLists), Gangyourex (вариация Meta) и другие.

**4.** DNS в настройках клиента не должен влиять на работоспособность. Ранее была замечена проблема, что из-за недоступности DNS от Cloudflare / Google / других, конфиги с доменными именами не работали до смены на Яндекс. Те конфиги все еще присутствуют в подписке, но в актуальной версии подписки все они сразу представлены с соответствующими IP-адресами и DNS не требуется вовсе.

**5.** При доступном интернете должны работать вплоть до 100% серверов. Скрипт проверяет работоспособность конфигов при обновлениях. Но, как уже упоминалось выше, прошу использовать ее только при необходимости из-за отключений интернета.
